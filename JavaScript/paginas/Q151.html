Q151. Qual declaração é aplicável ao atributo defer da tag HTML <script> ?


defer permite que o navegador continue processando a página enquanto o script é carregado em segundo plano.

defer faz com que o script seja carregado da rede de entrega de conteúdo (CDN) de backup.

defer bloqueia o navegador de processar o HTML abaixo da tag até que o script seja completamente carregado.

defer lazy carrega o script, fazendo com que ele seja baixado somente quando for chamado por outro script na página.
